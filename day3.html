Love Story お願い上手になろう

Pleaseの4つのパターン

📅 今日の目標

「Please」を使った丁寧なお願いの仕方を4つのパターンで覚えよう。これでもっと優しく、礼儀正しいお願いができるようになります！

👤 基本情報

💫 Pleaseの魔法の力

「Please」をつけるだけで、命令文がとても丁寧になります！

例：

❌ Help me!（助けて！）→ ⭐ Please help me.（お願いします、助けてください）

❌ Come here!（ここに来い！）→ ⭐ Please come here.（どうぞこちらへ来てください）

🎯 Pleaseの4つのパターン

パターン1: Please + 動詞

Please help me.（お願いします、助けてください）
Please wait.（お待ちください）
Please come here.（こちらに来てください）

パターン2: 動詞 + please

Help me, please.（助けてください、お願いします）
Wait, please.（待ってください、お願いします）
Come here, please.（来てください、お願いします）

パターン3: Please don't + 動詞

Please don't go.（お願い、行かないで）
Please don't worry.（どうぞ心配しないでください）
Please don't forget.（忘れないでくださいね）

パターン4: Don't + 動詞 + please

Don't go, please.（行かないで、お願い）
Don't worry, please.（心配しないで、お願い）
Don't forget, please.（忘れないで、お願い）

✍️ 練習してみよう
